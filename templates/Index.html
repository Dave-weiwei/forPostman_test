📘 書香小站 BookStation

✅ API 狀態檢查

伺服器目前狀態良好。

你可以使用 Postman 對以下 API 發送請求：

  • POST /register：註冊帳號
  • POST /login：登入（回傳 token）
  • GET /profile：查看個人資料（需登入）
  • GET /books?page=1&limit=10：查詢書籍（支援分頁）
  • POST /books：新增書籍（含封面）限管理員

請使用 Bearer Token 驗證授權。
